표 수정

표 수정은 표시 유형 및 형식을 피벗표에 적용합니다. 다음을 수행할 수 있습니다.
  • 개별 데이터 셀과 행 및 열 레이블의 모양 제어
  • 특정 열의 너비 설정 또는 특정 열 숨기기
  • 표 차원 피벗팅
이러한 수정 사항과 범위를 적용할 특정 유형의 표를 뷰어에서 선택할 수 있습니다.

•버전 19에서 사용할 수 있는 경량 표는 편집할 수 없으므로 이 명령의 영향을 받지 않습니다. 이후 버전에는 이 제한이 적용되지 않습니다.

선택 지정은 다음 항목으로 구성됩니다.

  • 처리할 표 유형. 유형은 OMS 표 하위 유형입니다.
  • 차원: 열 또는 행(기본값: 열)
  • 특정 행 또는 열(모두 포함)
  • 애플리케이션 범위: 선행 명령 또는 모든 결과(기본값: 선행 결과)
  • 선택적 값 관련 선택기준(예: 셀 값이 2보다 커야 함)
  • 유의성 표식이 있는 셀의 선택적 선택
동작 지정은 다음과 같습니다.
  • 지정된 행 또는 열 숨기기
  • 열 너비 및/또는 행 레이블 너비 설정
  • 글꼴 유형을 보통, 굵게, 기울임꼴 또는 굵은 기울임꼴로 설정
  • 글꼴 색상 설정
  • 배경색 설정
  • 지정된 셀에 사용자 정의 동작 적용
선택 지정은 기본 대화 상자와 유의성 하위 대화 상자에서 수행됩니다. 동작 지정은 동작 하위 대화 상자에서 수행됩니다. 조치를 지정하지 않거나 선택 기준이 일치하는 표를 찾지 못하면 명령은 자동으로 아무 작업도 수행하지 않습니다.

표, 차원 및 행 또는 열 선택

표 하위 유형 필드에 OMS 표 하위 유형을 입력하십시오. 하위 유형은 표 인스턴스에서 개요를 마우스 오른쪽 단추로 클릭하고 OMS 하위 유형 복사를 선택하여 찾거나 유틸리티>OMS 식별자의 목록에서 찾을 수 있습니다. 하위 유형을 모두 선택하려면 *를 입력하십시오.

열 또는 행을 선택하여 처리할 표의 차원을 결정합니다. 너비는 열 차원에 대해서만 설정할 수 있지만 행 레이블 너비는 차원에 관계없이 설정할 수 있습니다.

선택한 열 또는 행 처리할 특정 행 또는 열 목록을 공백으로 구분하여 입력하거나, 전체 차원을 처리하려면 "<<ALL>>"(따옴표 포함 및 모두 대문자)을 입력하십시오. 행 또는 열은 숫자 위치 또는 따옴표로 묶인 레이블 텍스트로 지정할 수 있습니다.

텍스트 매치 수준의 경우 첫 번째(가장 바깥쪽) 행 또는 열은 0입니다. 끝에서부터 거꾸로 계산하려면 음수를 사용하십시오. 레이블 텍스트 매치의 경우 텍스트를 따옴표로 묶으십시오. 레이블의 대소문자와 포함된 공백을 정확히 매치시키십시오. 숫자 및 텍스트 지정을 조합할 수 있습니다. 예를 들면 다음과 같습니다.
2 3 "상한"
이 경우 행 또는 열 2, 3 및 "상한" 레이블이 지정된 행 또는 열이 선택됩니다. 지정사항과 매치되는 항목이 없을 경우 무시됩니다. 행 또는 열은 표의 가장 안쪽 수준에서 계산됩니다.

위 텍스트가 패턴임을 선택할 경우 위의 각 텍스트 필드(<<ALL>> 제외)가 정규식으로 처리됩니다. 예를 들어 "Sig\."는 Sig.가 포함된 모든 레이블 텍스트와 매치됩니다. 그렇지 않은 경우 레이블은 지정된 텍스트와 문자 그대로 매치됩니다. 이것은 정규식으로서 Asympt Sig. 또는 Two-tailed Sig.와 매치되지만, Asympt Sig(마침표 없음)와는 매치되지 않습니다. 일반 텍스트인 경우 어느 것과도 일치하지 않습니다. 모든 Python 정규식을 사용할 수 있습니다. 패턴은 대/소문자를 구분하고 로케일을 인식합니다.

숫자처럼 보이는 항목은 선택란 설정에 관계없이 항상 위치로 처리됩니다.

정규식에 대한 전체 설명은 이 도움말의 범위를 벗어나지만 음수 표현식을 작성하는 기능은 언급할 필요가 있습니다. 예를 들어, "Total"로 시작하는 경우를 제외한 모든 행이나 열을 선택한다고 가정합니다. 이는 다음과 같이 작성할 수 있습니다.
"^(?!Total)" 캐럿(^)은 레이블 시작 부분에서 매치를 시작합니다. 나머지는 다음 텍스트가 Total이 아닌 경우에만 매치됩니다.

정규식은 데이터 열 너비 지정과 함께 사용할 수 없습니다.

텍스트 매치 시 기본적으로 가장 안쪽 수준(-1)의 텍스트가 사용됩니다. 밖으로 이동(위 또는 왼쪽)하려면 음수를 입력하고 바깥쪽에서 안쪽으로 계산하려면 양수를 입력하십시오. 일부 표에는 보이는 수준 사이에 보이지 않는 수준이 있으므로 수준을 올바르게 맞추려면 몇 가지 실험이 필요할 수 있습니다. 일반적으로 수준 1은 볼 수 있는 가장 바깥쪽 수준입니다.

선택한 차원의 모든 레이블을 보려면 레이블 인쇄를 선택하십시오. 대부분의 표에는 숨겨진 레이블이 있으므로 전체 구조를 보려면 이 옵션을 사용하십시오.

기본적으로 바로 앞의 프로시저 결과에 있는 지정된 유형의 모든 표가 처리됩니다. 이전에 실행된 프로시저를 모두 처리하려면 모든 결과를 선택하십시오.

유의성 하위 대화 상자

이 하위 대화 상자는 기본 대화 상자에서 선택한 항목을 여기에 지정된 유의성 기준을 충족하는 셀로 좁힙니다. 대부분의 경우 유의성 기준은 기본 대화 상자의 "<<ALL>>" 선택 지정과 함께 사용되지만, 거기에 선택된 부분집합이 있는 경우 그 안에 있는 유의성 지정이 적용됩니다.

•이 하위 대화 상자는 유의성 표식이 기본 표에 표시되고 단순 유형(APA 아님)인 사용자 정의 표(CTABLES)의 결과 및 Statistics 버전 24 이상에만 적용됩니다.

유의한 셀 선택: 유의성 표식이 있는 셀을 모두 선택하려면 유의성 표식이 있는 모든 셀을 선택하십시오. 유의성 표식은 셀에 표시되는 특수 문자 코드입니다.

또는 유의성 패턴을 입력하여 선택할 특정 유의성 표식을 지정할 수 있습니다. 패턴은 단순한 열 표식 목록일 수 있습니다. 예를 들어 "AB"는 A 또는 B 열이 유의적으로 다른 셀만 선택합니다. 유의성 결과가 별도의 표에 표시되지 않고 병합된 경우 문자 코드가 열 머리말에 표시됩니다. 몇 가지 특정 범주에 초점을 맞추고 싶은 경우 이를 사용할 수 있습니다.

열 차원에 성별 내 직업 범주와 같은 중첩 지정이 있거나(gender > jobcat) 여러 범주형 변수가 해당 차원에 누적되어 있는 경우(jobcat + gender) 열 전체에 걸쳐 부표가 표시됩니다. 각 문자 코드 뒤에 부표 번호를 나열하여 선택사항이 적용될 부표를 추가로 지정할 수 있습니다. 예를 들어 A01B2는 부표 0과 1에서 A 유의 셀만 선택하고 부표 2에서 B 유의 코드만 선택해야 함을 나타냅니다. 유의성에 대한 열 문자는 표 전체에서 부표마다 (A)부터 다시 시작되므로 결과에서 부표를 식별할 수 있습니다.

중첩된 표의 경우 가장 안쪽 차원이 반복되므로 중첩된 모든 부표에 선택 항목을 적용할 수 있습니다. 그러나 변수가 여러 열에 걸쳐 누적되어 있는 경우 유의성 문자의 의미가 부표마다 다르므로 선택 항목을 특정 부표에만 적용할 수 있습니다.

• 부표 번호는 0부터 시작됩니다. 이 표기법을 사용하여 최대 10개의 부표를 참조할 수 있습니다.

유의수준: 하나 또는 두 개의 유의수준이 표에 표시될 수 있습니다. 두 개가 있는 경우 더 유의한 값은 대문자로 표시되고 덜 유의한 값은 소문자로 표시됩니다. 선택할 때 두 수준을 모두 선택하거나 둘 중 하나만 선택하도록 지정할 수 있습니다. 표에 수준이 하나만 있는 경우 둘 다 또는 상한을 선택하십시오.

동작 하위 대화 상자

이러한 항목을 보이지 않게 하려면 선택한 행 또는 열 숨기기를 선택하십시오. 이 옵션은 다른 동작과 함께 사용할 수 없습니다.

열 너비 설정

열 너비 필드에 선택한 모든 열에 적용할 숫자를 포인트 단위로 입력하거나 선택된 열 수에 해당하는 숫자 목록을 입력하여 개별 너비를 지정하십시오. 1포인트는 1/72인치입니다. 힌트: 열의 현재 너비를 확인하려면 표를 두 번 클릭하여 활성화하고 열 테두리를 클릭한 후 살짝 끕니다.

행 레이블 너비는 행 또는 열과 단일 너비 또는 너비 세트를 선택할 때와 동일한 표기법으로 행 레이블 위치를 입력하여 제어할 수 있습니다.

유형 적용

유형은 표의 데이터 셀, 선택한 차원의 레이블 또는 둘 다에 적용할 수 있습니다. 조건을 충족하는 데이터 셀만 처리하려면 조건의 데이터 셀(기본 대화 상자에 있음)을 선택하십시오.

조건은 셀 값이 x로 표시되는 true/fals 표현식입니다. 예를 들면 다음과 같습니다.

x < .05
이 경우 셀 값이 .05보다 작은 데이터 셀만 선택됩니다. 표현식은 Python 코드로 해석됩니다. 일반 논리 및 비교 연산자를 사용할 수 있습니다. abs(x)는 셀의 절대값입니다.

i는 SELECT 차원의 반대 차원에 있는 행 또는 열 번호를 나타냅니다. 예를 들면 다음과 같습니다.
i % 2 == 1
(%는 Python 모드 연산자입니다.) 이 경우 선택한 차원이 행이면 홀수 열이 선택됩니다.

ii는 SELECT 차원의 행 또는 열을 나타냅니다. 다음을 지정할 경우
ii % 2 == 1
SELECT 차원이 행이면 홀수 행에 유형을 적용하는 데 사용될 수 있습니다.

해당 로케일에서 일반 소수점 기호가 콤마인 경우에도 항상 점(.)을 사용하여 소수점 값을 지정하십시오.

적절한 제어를 사용하여 텍스트 유형, 색상 및 배경색을 선택하십시오. 색상은 세 개의 숫자로 표시되는데, 각각 빨간색, 녹색, 파란색 성분에 해당합니다. 값은 0(없음) 및 255(최대값) 사이여야 합니다. 예를 들어, 0 0 0은 검은색, 255 255 255는 흰색, 255 0 0은 빨간색입니다. 노란색은 빨간색 및 녹색의 동일한 부분으로 구성됩니다.

지정된 셀에 적용할 사용자 정의 함수가 있는 경우 여기에 다음과 같이 입력하십시오.
modulename.functionname
이러한 함수를 작성하는 방법에 대한 자세한 내용과 예제는 Python 모듈 customstylefunctions.py를 참조하십시오.

사용자 정의 함수에는 매개변수가 있을 수 있습니다. 하나 이상의 매개변수를 지정하려면 매개변수 목록을 괄호로 묶고 매개변수 이름과 값을 작성하십시오. 예를 들면 다음과 같습니다.
customstylefunctions.stripeOddRows2(r=20,g=20,b=200)
이 경우 stripeOddRows2 함수가 customstylefunctions 모듈에 적용됩니다. 세 개의 매개변수는 각각 빨간색, 녹색 및 파란색을 나타내는 r, g, b이며 값은 각각 20, 20 및 200입니다. 허용되는 매개변수를 확인하려면 특정 사용자 정의 함수에 대한 문서를 참조하십시오. 매개변수 이름은 함수 정의에 지정된 대소문자로 작성해야 합니다.

매개변수 값이 문자열이면 다음과 같이 작은따옴표로 묶으십시오.
option='both'

표 모양

표 모양 적용을 선택하고 표 모양 파일을 지정하여 전체 표에 표 모양을 적용할 수 있습니다. 표 모양 파일의 확장자는 stt입니다. 표 모양은 기본 대화 상자의 설정에 관계없이 전체 표에 적용됩니다.

올바른 색상 선택

프레젠테이션에 가장 적합한 색상은 목적과 프리젠테이션이 표시되는 장치의 특성에 따라 다릅니다. 항상 특정 행이나 열을 강조하거나 일반적이거나 특이한 값을 강조할 수 있습니다. 최상의 색상 구성표는 화면에서 볼 것인지, 인쇄할 것인지, 투영할 것인지에 따라 다를 수 있습니다. 또는 회사 색상 구성표가 필요할 수도 있습니다. 다음에서 조언과 RGB 좌표를 얻을 수 있습니다.
http://www.personal.psu.edu/cab38/ColorBrewer/ColorBrewer.html .
이 사이트는 지도에 관한 것이지만 일부 조언은 표(또는 도표)에도 유용할 수 있습니다.

추가 기능

이 대화 상자는 SPSSINC MODIFY TABLES 확장 명령에 대한 구문을 생성합니다. 이 명령에 대한 도움말을 표시하려면 다음 구문을 실행하십시오.

SPSSINC MODIFY TABLES /HELP.

명령 명령문에서 따옴표로 묶은 개별 하위 유형의 목록을 지정할 수 있습니다.

요구 사항

이 대화 상자를 사용하려면 Integration Plug-In for Python과 SPSSINC MODIFY TABLES 확장 명령문이 필요합니다.
